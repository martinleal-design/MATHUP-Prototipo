Enciclopedia Geométrica PAES

Todas las fórmulas clave de figuras 2D y cuerpos 3D que necesitas dominar.

Cuadrado

  • Área: A = a²
  • Perímetro: P = 4a
  • Diagonal: d = a√2

Rectángulo

  • Área: A = a·b
  • Perímetro: P = 2a + 2b
  • Diagonal: d = √(a² + b²)

Triángulo

  • Área: A = (b·h)/2
  • Perímetro: P = a + b + c
  • Pitágoras: c² = a² + b²

Círculo

  • Área: A = π·r²
  • Perímetro: P = 2πr
  • Diámetro: d = 2r

Cubo (3D)

  • Volumen: V = a³
  • Área Total: At = 6a²
  • Diagonal: d = a√3

Cilindro (3D)

  • Volumen: V = π·r²·h
  • Área Basal: Ab = 2πr²
  • Área Total: At = 2πr(h+r)

Cono (3D)

  • Volumen: V = (π·r²·h)/3
  • Generatriz: g = √(r²+h²)
  • Área Total: At = πr(g+r)

Esfera (3D)

  • Volumen: V = (4/3)π·r³
  • Área Sup.: A = 4πr²
  • Hemisferio: V = (2/3)π·r³
